# Polyglot Tic-Tac-Toe

独立した14のプログラミング言語で動作する、GitHub Actionsベースの三目並べ技術デモです。

### 遊び方

以下のボード上の空いているマス（`___`）をクリックして、Issueを作成してください。
各言語専用のGitHub Actionsが自動的に起動し、あなたの手を処理します。

- **C++**: `.github/workflows/cpp.yml`
- **C#**: `.github/workflows/csharp.yml`
- **Go**: `.github/workflows/go.yml`
